# Phase 4 테스트 가이드: 게시판 + 상담

```bash
cd c:\Users\user\Documents\aafc
python manage.py runserver
```

## 2. 게시판 목록 테스트

### 2-1. 공지사항
- [ ] `http://127.0.0.1:8000/board/Y/` 접속

- [ ] 상단에 [공지] 배지 붙은 공지사항 고정 표시
- [ ] 일반 글에 [공지]/[소식]/[이벤트] 카테고리 라벨 표시
- [ ] 댓글 수 [N] 표시되는지 확인
- [ ] 페이지네이션 동작 (15건씩)

### 2-2. 포토갤러리
- [ ] `http://127.0.0.1:8000/board/P/` 접속
- [ ] 그리드(사진) 형식 목록 표시
- [ ] 본문의 첫 번째 이미지가 썸네일로 표시되는지 확인
- [ ] 제목 클릭 시 상세 페이지 이동

### 2-3. 학부모 다이어리
- [ ] `http://127.0.0.1:8000/board/PR/` 접속

- [ ] 글쓰기 버튼 표시 (로그인 시)

### 2-4. 공부하는 AAFC
- [ ] `http://127.0.0.1:8000/board/ST/` 접속
- [ ] 테이블 형식 목록 정상 표시

### 2-5. 검색 기능
- [ ] 공지사항 목록에서 검색어 입력 후 검색
- [ ] 제목 또는 내용에 검색어가 포함된 글만 표시되는지 확인

## 4. 게시글 상세 테스트

### 4-1. 일반 게시글 상세
- [ ] 공지사항 목록에서 아무 글 클릭
- [ ] 제목, 작성자, 작성일, 조회수 표시
- [ ] 본문 HTML 정상 렌더링
- [ ] 이전글/다음글 네비게이션 표시
- [ ] 목록 버튼 클릭 시 목록으로 이동

### 4-2. 첨부파일 게시글
- [ ] 첨부파일이 있는 게시글 확인
- [ ] 이미지 첨부파일 표시
- [ ] 다운로드 파일 링크 표시

### 4-3. 조회수
- [ ] 게시글 조회 시 조회수 1 증가 확인
- [ ] 본인 글은 조회수 증가하지 않는지 확인

### 4-4. 클래식반 (U8/U10/U12)
- [ ] `http://127.0.0.1:8000/board/U8/199/` 접속
- [ ] 아카데미 서브메뉴(구장안내/코칭스태프/교육프로그램/클래식반) 표시
- [ ] U-8/U-10/U-12 탭 표시 및 전환
- [ ] 댓글/이전다음글 영역 없음 (단일 페이지)

## 5. 댓글 테스트

### 5-1. 댓글 등록
- [ ] 로그인 상태에서 게시글 상세 페이지 접속
- [ ] 댓글 입력란에 텍스트 입력 후 "등록" 클릭
- [ ] "댓글등록하시겠습니까?" 확인 팝업
- [ ] 등록 후 댓글 목록에 표시 (작성자명, 아이디, 작성일)

### 5-2. 댓글 삭제
- [ ] 본인이 작성한 댓글 옆 삭제 버튼 표시
- [ ] 삭제 클릭 → "삭제하시겠습니까?" 확인 팝업
- [ ] 삭제 후 댓글 목록에서 제거

### 5-3. 비로그인 댓글
- [ ] 비로그인 상태에서 "등록" 클릭 시 로그인 페이지로 이동

---

## 6. 글쓰기 테스트 (학부모 다이어리 / 포토갤러리)

### 6-1. 글쓰기 접근
- [ ] 로그인 후 학부모 다이어리 목록에서 "글쓰기" 버튼 클릭
- [ ] 글쓰기 폼 표시 (제목 입력, CKEditor 에디터)
- [ ] 작성자명 자동 표시

### 6-2. CKEditor 5 동작
- [ ] CKEditor 에디터 로드 (툴바: 제목, 볼드, 이탤릭, 링크, 리스트 등)
- [ ] 텍스트 서식 적용 (볼드, 이탤릭 등)
- [ ] 링크 삽입 가능

### 6-3. 글 등록
- [ ] 제목, 내용 입력 후 "등록" 클릭
- [ ] "등록하시겠습니까?" 확인 팝업
- [ ] 등록 후 목록 페이지로 이동
- [ ] 등록한 글이 목록 최상단에 표시

### 6-4. 제목 없이 등록
- [ ] 제목 비우고 등록 시 "제목을 입력하세요" 알림

### 6-5. 비로그인 글쓰기
- [ ] 비로그인 상태에서 글쓰기 버튼 → 로그인 페이지로 이동

## 7. 글 수정/삭제 테스트

### 7-1. 글 수정
- [ ] 본인 글 상세 페이지에서 "수정" 버튼 표시
- [ ] 수정 페이지에서 기존 제목/내용 로드 확인
- [ ] 내용 수정 후 저장
- [ ] 상세 페이지에서 수정 내용 반영 확인

### 7-2. 글 삭제
- [ ] 본인 글 상세 페이지에서 "삭제" 버튼 표시
- [ ] 삭제 클릭 → "삭제하시겠습니까?" 확인 팝업
- [ ] 삭제 후 목록에서 해당 글 제거 (soft delete)

### 7-3. 타인 글
- [ ] 다른 사용자의 글에서 수정/삭제 버튼 없음 확인

## 8. 상담신청 테스트

### 8-1. 상담 페이지 접속
- [ ] `http://127.0.0.1:8000/consult/` 접속
- [ ] sub_top 이미지 (상담) 표시
- [ ] 서브메뉴: "상담신청"(활성), "무료수업체험 신청" 표시

### 8-2. 개인정보 동의
- [ ] "동의" 라디오 미선택 시 등록 → "개인정보 수집/이용에 동의하셔야..." 알림

### 8-3. 지역 검색 → 구장 선택
- [ ] 지역명 입력 (예: "마포구") 후 검색 버튼
- [ ] AJAX로 구장 목록 로드 (라디오 버튼)
- [ ] 구장 미선택 시 → "구장을 선택하여 주세요" 알림

### 8-4. 상담 폼 입력
- [ ] 상담유형 드롭다운 (공통코드 CONT에서 로드)
- [ ] 신청자명, 연락처(3분할), 수강생명, 성별, 나이, 제목, 내용 입력
- [ ] 필수 항목 비어있을 때 각각 알림 메시지

### 8-5. 상담 등록
- [ ] 모든 항목 입력 후 "상담신청" 클릭
- [ ] "온라인상담 등록하시겠습니까?" 확인 팝업
- [ ] 등록 완료 페이지: "상담이 접수되었습니다" 메시지
- [ ] Django Admin에서 해당 상담 건 확인

## 11. Django Admin 테스트

### 11-1. 관리자 접속

### 11-2. 게시판 관리

### 11-5. 무료체험 관리
