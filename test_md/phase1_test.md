# Phase 1 테스트 가이드: 회원 + 인증

## 사전 준비
```bash
cd c:\Users\user\Documents\aafc
python manage.py runserver
```
브라우저에서 `http://127.0.0.1:8000` 접속

---

## 1. 데이터 이관 검증

### 1-1. 회원 수 확인
```bash
python manage.py shell -c "from apps.accounts.models import Member, MemberChild, OutMember; print(f'Member: {Member.objects.count()}건, MemberChild: {MemberChild.objects.count()}건, OutMember: {OutMember.objects.count()}건')"
```
- [ ] Member: 10,150건 (+ admin 1건 = 10,151건)
- [ ] MemberChild: 11,429건
- [ ] OutMember: 35건

### 1-2. 샘플 데이터 확인
```bash
python manage.py shell -c "from apps.accounts.models import Member; m = Member.objects.get(username='hanz811'); print(f'이름: {m.name}, 전화: {m.phone}, 이메일: {m.email}')"
```
- [ ] 이름, 전화번호, 이메일이 정상적으로 출력되는지

---

## 2. 로그인 테스트

### 2-1. 기존 회원 로그인 (SHA256 호환)
- [ ] `http://127.0.0.1:8000/accounts/login/` 접속
- [ ] 기존 MSSQL 회원 아이디/비밀번호로 로그인 시도
- [ ] 로그인 성공 시 헤더에 "로그아웃", "마이아카데미" 표시되는지 확인

### 2-2. 잘못된 비밀번호
- [ ] 아이디는 맞고 비밀번호 틀리게 입력
- [ ] 에러 메시지 표시되는지 확인

### 2-3. 로그아웃
- [ ] 헤더의 "로그아웃" 클릭
- [ ] 헤더가 "로그인", "회원가입"으로 변경되는지 확인

---

## 3. 회원가입 테스트

### 3-1. 신규 가입
- [ ] `http://127.0.0.1:8000/accounts/register/` 접속
- [ ] 아이디, 비밀번호, 비밀번호 확인, 이름, 휴대전화 입력
- [ ] 가입 완료 후 자동 로그인되는지 확인

### 3-2. 중복 아이디
- [ ] 이미 존재하는 아이디로 가입 시도
- [ ] 에러 메시지 표시되는지 확인

### 3-3. 비밀번호 불일치
- [ ] 비밀번호와 비밀번호 확인을 다르게 입력
- [ ] 에러 메시지 표시되는지 확인

---

## 4. 마이페이지 테스트

### 4-1. 마이페이지 접근
- [ ] 로그인 상태에서 `http://127.0.0.1:8000/accounts/mypage/` 접속
- [ ] 회원명, 아이디, 휴대폰, 이메일, 주소, SMS/메일 수신동의 표시되는지 확인
- [ ] 자녀 정보 (이름, 생년월일, 학교, 성별) 표시되는지 확인

### 4-2. 비로그인 접근
- [ ] 로그아웃 상태에서 마이페이지 URL 직접 입력
- [ ] 로그인 페이지로 리다이렉트되는지 확인

---

## 5. 프로필 수정 테스트

### 5-1. 정보 수정
- [ ] 마이페이지 → "정보수정" 버튼 클릭
- [ ] 이름, 이메일, 전화번호 등 수정
- [ ] 저장 후 마이페이지에서 변경된 내용 확인

---

## 6. 자녀 관리 테스트

### 6-1. 자녀 등록
- [ ] 마이페이지 → "자녀등록" 버튼 클릭
- [ ] 이름, 생년월일, 성별, 학교, 학년 입력
- [ ] 저장 후 마이페이지에서 등록된 자녀 확인

### 6-2. 자녀 수정
- [ ] 자녀 정보 옆 "정보수정" 버튼 클릭
- [ ] 정보 수정 후 저장
- [ ] 마이페이지에서 변경 확인

### 6-3. 자녀 삭제 (신규 등록한 자녀로 테스트)
- [ ] 자녀 삭제 페이지에서 "삭제" 클릭
- [ ] 마이페이지에서 삭제되었는지 확인

---

## 7. 비밀번호 변경 테스트

### 7-1. 비밀번호 변경
- [ ] 마이페이지 → "비밀번호 변경" 클릭
- [ ] 현재 비밀번호, 새 비밀번호, 새 비밀번호 확인 입력
- [ ] 변경 후 새 비밀번호로 로그인 가능한지 확인

### 7-2. 현재 비밀번호 틀림
- [ ] 현재 비밀번호를 틀리게 입력
- [ ] 에러 메시지 표시되는지 확인

---

- [ ] 회원 상세 페이지에서 정보 확인

### 8-3. 자녀 관리

- [ ] 수강상태, 성별 필터 동작

### 8-4. 탈퇴회원
- [ ] 탈퇴회원 목록 표시되는지 확인
